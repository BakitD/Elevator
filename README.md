# Elevator


### Installation

> `git clone git@github.com:BakitD/Elevator.git`

### Run

> `cd Elevator`

To run program

> `python src/main.py -floors FLOORS -height HEIGHT -speed SPEED -wait_time`

To specify arguments possible values

> `python src/main.py -h`


To run tests

> `python -m unittest tests/*.py`
